# Personal Resume Website

This repository contains my personal resume website, built using HTML, Tailwind CSS, and GitHub Pages.

## Live Website

View the live site here:
https://christopherthrasher.github.io/resume-site/

## Overview

This website includes:

- Landing page (index.html)
- About page (about.html)
- Experience page (experience.html)

The site is designed to provide a clean, professional overview of my background, experience, and career interests.

## Technologies Used

- HTML
- Tailwind CSS (via CDN)
- Git
- GitHub
- GitHub Pages

## Purpose

This website was built to:

- Create a professional online presence
- Showcase experience and leadership
- Provide a shareable portfolio link for resumes and applications

## Deployment

The site is deployed using GitHub Pages directly from the main branch.

Any updates pushed to the repository automatically update the live site.
